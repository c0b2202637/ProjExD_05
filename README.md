# 走れこうかとん
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
横スクロールアクション

## ゲームの実装
###共通基本機能
* 主人公キャラクターと地面に関するクラス

### 担当追加機能
* 障害物が動いてくる事に関するクラス　福山
* ジャンプする機能　大久保
* スキル１のクラス（空飛ぶ機能）ろじゅんごう : tabキーを押す時、落下を止める
* スキル２のクラス（星をとると無敵）　こうき
* スキル３のクラス（障害物を破壊する）福山
