# 走れこうかとん
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
横スクロールアクション

## ゲームの実装
###共通基本機能
* 主人公キャラクターと地面に関するクラス

### 担当追加機能
* 障害物に関するクラス　大久保
* スコア　なかざわ
* ※地形生成に関するクラス
* スキル１のクラス（空飛ぶ）ろじゅんごう
* スキル２のクラス（無敵）　こうき
* スキル３のクラス（障害物破壊）福山
